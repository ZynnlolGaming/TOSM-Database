# 🌿 Tree of Savior M — Database

A fan-made database website for **Tree of Savior M** that makes it easy to search and browse game information in one place.

🔗 **Live Website:** [https://zynnlolgaming.github.io/TOSM-Database](https://zynnlolgaming.github.io/TOSM-Database)

---

## 📖 What's Inside

| Section | Description |
|---|---|
| ⚔️ **Monsters** | Browse all monsters by type, element, rank, and episode. Includes in-game portraits. |
| 👥 **Fellow** | Full Fellow database with tier ratings, acquire bonus, and grade filters. Includes Demigod portraits. |
| 🏪 **Kupole** | Kupole database with rank bonus, passive/active skills, and tier ratings. |
| 🃏 **Monster Card** | Monster Card database with effects at Lv.1, Lv.10, and Lv.20, drop sources, rarity, and episode filters. |

---

## ✨ Features

- 🔍 Search by name across all sections
- 🎛️ Filter by type, element, grade, rarity, episode, and acquire bonus
- 🖼️ In-game portraits for monsters, Demigod fellows, Kupoles, and Monster Cards
- 🌙 Dark mode / ☀️ Light mode toggle
- 📱 Works on desktop and mobile
- 🗺️ ToS world map as background
